# Attention Layer Plan

## Goal
Make the dashboard more proactive and capable of getting Christian's attention when something matters.

## Mechanisms
- toast notifications
- badges
- banners
- persistent alert cards
- escalating reminders for truly important items

## Good triggers
- important email
- prep needed for a meeting
- meeting starting soon
- uncaptured post-meeting actions
- urgent overdue task
- real blocker

## Bad triggers
- promos
- duplicate status noise
- low-value chatter

## Rule
Attention should be intelligent, not spammy.
