# Daily Digest Dashboard Module

## Goal
Create a persistent executive digest surface in the dashboard so daily context is always visible and not lost in chat.

## Module sections
- today / tomorrow calendar
- important prep needed
- top tasks/reminders
- important emails
- meeting follow-ups waiting
- blockers / decisions

## Rules
- digest lives in dashboard first
- chat is a nudge layer, not the only home
- each digest item should link back to the host app/site when possible

## Desired actions
- open Gmail
- open calendar event / meeting link
- open task/reminder source
- open relevant docs
