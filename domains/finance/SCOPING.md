# Finance Scoping Rule

To avoid mixing finance with generic documents:

## Canonical storage rule
Finance documents live only under `/Volumes/Storage/OpenClaw/domains/finance/`.

Do **not** use a generic `Documents/` root for finance.

## Finance includes
- invoices
- receipts
- statements
- billing notices
- reimbursement docs
- tax support docs
- payroll/401k/retirement-plan billing and termination paperwork
- unauthorized withdrawal evidence
- vendor payment/cancellation correspondence when tied to money movement

## Non-finance documents should live elsewhere
Examples:
- contracts not tied to billing workflow
- design files
- marketing assets
- client project deliverables
- general admin docs

## Retrieval rule
When looking for a money-related document, search Finance first.
When looking for non-money documents, ignore Finance unless explicitly told otherwise.
