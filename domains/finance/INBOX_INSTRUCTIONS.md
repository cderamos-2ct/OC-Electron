# Finance Inbox Instructions

Drop files here when they need finance intake.

## Use these folders
- `Inbox/manual/` — PDFs, receipts, statements, screenshots, vendor docs
- `Inbox/mobile/` — phone photos or scans
- `Inbox/email/` — reserved for automatically collected email attachments

## What belongs here
- invoices
- receipts
- vendor statements
- reimbursement docs
- bank-charge evidence
- cancellation notices
- retirement-plan admin docs tied to billing/issues

## What does NOT belong here
- contracts with no current finance action
- random general business docs
- creative/project files
- personal non-finance docs

## Processing behavior
Items dropped here can be:
1. fingerprinted
2. OCR/text-extracted
3. classified
4. added to the finance database
5. moved into canonical storage under `Documents/YYYY/MM/`

If extraction is uncertain, files should go to `Review/`.
