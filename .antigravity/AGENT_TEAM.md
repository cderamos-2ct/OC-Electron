# Agent Team Management

## Team Shape
1. lead-architect
2. planner
3. executor
4. reviewer
5. security-reviewer
6. qa-visual
7. release-manager

## Operating Rules
1. Planner breaks work into executable tasks with dependencies.
2. Architect approves scope boundaries for each active task.
3. Executor owns implementation.
4. Reviewer/Security/QA validate before release-manager gate.
5. No merge-ready status without gate pass.
