## Summary
Extend the dashboard beyond command chat with a dedicated Personal Ops / Comms visibility layer so Christian can understand and steer life-admin/comms work across Gmail, Messages, Calendar, Reminders, Notes, and meeting recap intake without digging through raw apps or worker traces.

## Acceptance
- [ ] Dashboard exposes a trustworthy unified attention view across comms/personal-ops sources.
- [ ] Christian can see bucketed inbox state (`urgent`, `needs reply`, `follow-up`, `waiting`, `reference`, `junk`) instead of just raw unread counts.
- [ ] Recent meetings and meeting recaps are tied to follow-ups/tasks rather than living as isolated emails.
- [ ] Reminders and notes can be surfaced as actionable context next to comms/tasks, not as separate hidden silos.
- [ ] Drill-ins show source evidence (email/message/reminder/note/event) without forcing raw runtime transcripts as the default surface.
- [ ] Command chat remains the orchestration surface, but visibility-heavy work has richer companion views.

## Proposed Dashboard Surface Model
### 1. Attention Center
Top-level executive queue for the day:
- Needs reply now
- Waiting on others
- Meetings today / next
- Follow-ups due back
- New recaps / notes to process
- Risks / overdue / stalled

### 2. Comms Inbox Board
Unified triage board across Gmail + Messages with explicit buckets:
- Urgent
- Needs reply
- Follow-up
- Waiting
- Reference
- Junk

Each item should show:
- source (Gmail, Messages, Fireflies, etc.)
- human / sender
- subject / snippet
- account or chat
- latest timestamp
- current owner
- linked task / reminder / draft state
- suggested next action

### 3. Calendar + Meeting Follow-up View
Meeting-centric operational surface:
- today / upcoming meetings
- prep needed
- recap received / missing
- extracted action items
- unassigned follow-ups
- attendees / linked threads / related notes

### 4. Notes + Reminders Context Rail
Side panel for non-chat context that still matters operationally:
- active reminders by list
- notes recently touched or matched to active work
- unresolved note-derived tasks
- ability to pin notes/reminders to tasks and conversations

### 5. Task Promotion / Handoff Controls
Fast controls on every item:
- create task
- attach to existing task
- mark waiting
- mark follow-up date
- prep reply draft
- send to agent lane
- archive/reference/junk

### 6. Source Drill-Down Drawer
Secondary evidence drawer for full context:
- full email body / attachments
- message thread history
- calendar invite details
- reminder notes
- note text / folder
